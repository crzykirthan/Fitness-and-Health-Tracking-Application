**🏋️ Fitness and Health Tracking Application**

A web-based Fitness and Health Tracking System that helps users monitor their fitness progress, set health goals, and connect with professional trainers.
The system allows users to manage their profiles, track progress, book trainers, and receive personalized recommendations.

**📌 Project Overview**

The Fitness and Health Tracking Application is designed to help individuals maintain a healthy lifestyle by tracking fitness progress and interacting with trainers.

Users can:

Register and log into the system

Update their personal profile

Set fitness goals

Track their progress

Book trainers

Receive trainer recommendations

Review and rate trainers

The application also includes trainer and admin functionalities for managing booking requests and monitoring user activity.

**✨ Features
👤 User Module**

User Registration and Login

Profile Management

Update personal details

Set and view fitness goals

Track health progress

**🏋️ Trainer Module**

View trainer booking requests

Accept or reject requests

Manage assigned clients

**📅 Trainer Booking System**

Users can book trainers

Only one trainer can be booked at a time

Trainer approval system

Booking status tracking

**🎯 Goal Management**

Users can set fitness goals

Goals are displayed in the profile page

Trainer recommendations based on goals

**⭐ Review and Rating System**

Users can review trainers

Ratings help other users choose trainers

**🔔 Notification System**

Booking status updates

Trainer acceptance notifications

**⚙ Installation Guide**

1️⃣ Clone the Repository

git clone https://github.com/crzykirthan/Fitness-and-Health-Tracking-Application.git

2️⃣ Move Project to XAMPP

Copy the project folder to:

C:\xampp\htdocs\
3️⃣ Start XAMPP

Start the following services:

Apache

MySQL

4️⃣ Import the Database

Open phpMyAdmin

Create a database:

fitness_db

Import the provided SQL file.

5️⃣ Run the Application

Open in browser:

http://localhost/fitness_website/frontend/index.html
